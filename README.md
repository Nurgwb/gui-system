# gui-system
单机网站开发平台

Springboot作为基础框架，使用mybatis作为持久层框架
使用官方推荐的thymeleaf做为模板引擎，shiro作为安全框架
内置代码生成器


定位于快速二次开发管理网站，中小型网站，restfull接口
